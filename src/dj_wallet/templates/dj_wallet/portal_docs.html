Wallet

MethodPathUse

GET/wallet/transactionsTransaction list

Signed Actions

For withdraw/transfer, sign this payload with the holder key:

{
  "holder_id": <USER_ID>,
  "holder_type": "auth.user",
  "action": "transfer",
  "amount": "1.00000000",
  "nonce": "<NONCE>"
}

Then send:

POST /api/wallet/transfer
Authorization: Token <TOKEN>
{"to_user_id":1,"amount":"1.00000000","nonce":"...","signature":"0x...","key_id":"user-0x..."}
