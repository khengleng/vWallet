# Tenant Onboarding Checklist

Use this checklist when enabling a new tenant.

## Pre‑Approval

1. Company details collected
2. Admin user created
3. Tenant status set to `pending`
4. KYC/KYB documents received

## Approval

1. Compliance review completed
2. Risk scoring assigned
3. Approval decision recorded
4. Tenant status set to `approved`

## Activation

1. Tenant status set to `active`
2. Default roles created (`customer`, `agent`, `ops`)
3. Customer limits applied
4. Anchor settings configured (chain/contract)
5. Branding set (name, logo, theme)
6. Support email configured

## Post‑Activation Validation

1. Test login for tenant admin
2. Test mobile PWA login for customer
3. Test cash in/out flow
4. Confirm anchors are being submitted
5. Verify audit hash and signatures
